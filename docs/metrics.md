# Evaluation Metrics

- **CLIP-I ↑** — CLIP image similarity. Higher is better.
- **FID ↓** — Fréchet Inception Distance. Lower is better.
- **BLEURT ↑** — BLEURT learned text similarity (for alt-text). Higher is better.
- **Fairness Δ ↓** — max pairwise disparity across protected groups. Lower is better.
